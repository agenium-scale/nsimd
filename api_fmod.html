Floating-point remainder

Description

Compute the floating-point remainder of its argument. For more informations visit https://sleef.org/purec.xhtml.

C base API (generic)

#define vfmod(a0, a1, type)
#define vfmod_e(a0, a1, type, simd_ext)

C advanced API (generic, requires C11)

#define nsimd_fmod(a0, a1)

C++ base API (generic)

template <NSIMD_CONCEPT_VALUE_TYPE T> typename simd_traits<T, NSIMD_SIMD>::simd_vector NSIMD_VECTORCALL fmod(typename simd_traits<T, NSIMD_SIMD>::simd_vector a0, typename simd_traits<T, NSIMD_SIMD>::simd_vector a1, T);

C++ advanced API

template <NSIMD_CONCEPT_VALUE_TYPE T, NSIMD_CONCEPT_SIMD_EXT SimdExt> pack<T, 1, SimdExt> fmod(pack<T, 1, SimdExt> const& a0, pack<T, 1, SimdExt> const& a1);
template <NSIMD_CONCEPT_VALUE_TYPE T, int N, NSIMD_CONCEPT_SIMD_EXT SimdExt> pack<T, N, SimdExt> fmod(pack<T, N, SimdExt> const& a0, pack<T, N, SimdExt> const& a1);
