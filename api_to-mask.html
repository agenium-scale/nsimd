Build mask from logicals

Description

Returns a mask consisting of all ones for true elements and all zeros for false elements.

C base API (generic)

#define vto_mask(a0, type)
#define vto_mask_e(a0, type, simd_ext)

C advanced API (generic, requires C11)

#define nsimd_to_mask(a0)

C++ base API (generic)

template <NSIMD_CONCEPT_VALUE_TYPE T> typename simd_traits<T, NSIMD_SIMD>::simd_vector NSIMD_VECTORCALL to_mask(typename simd_traits<T, NSIMD_SIMD>::simd_vectorl a0, T);

C++ advanced API

template <NSIMD_CONCEPT_VALUE_TYPE T, NSIMD_CONCEPT_SIMD_EXT SimdExt> pack<T, 1, SimdExt> to_mask(packl<T, 1, SimdExt> const& a0);
template <NSIMD_CONCEPT_VALUE_TYPE T, int N, NSIMD_CONCEPT_SIMD_EXT SimdExt> pack<T, N, SimdExt> to_mask(packl<T, N, SimdExt> const& a0);
